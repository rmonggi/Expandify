# Expandify

A lightweight, no-fuss text expansion utility for Windows. Define keyboard shortcuts that instantly expand into full text, code snippets, or rich content.

## 🚀 Features

- ⚡ **Instant Expansion** - Type a trigger, get your snippet instantly
- 📝 **Rich Text Support** - Expand to formatted text, HTML, or plain text
- 🖼️ **Embed Images** - Include images in your snippets
- 🎯 **Allowed Apps** - Choose which apps text expansion works in
- 🔒 **Smart Protection** - Auto-detects password fields, won't expand there
- ⌨️ **Global Keyboard Listener** - Works across all your applications
- 💾 **Auto-backup** - Your snippets are always safe
- 🎨 **Clean UI** - Minimal, intuitive interface

## 💡 Use Cases

- Quickly insert email signatures
- Expand code snippets in your IDE
- Insert frequently used text templates
- Generate boilerplate code
- Speed up repetitive typing

## 🛠️ Built With

- Electron
- Node.js
- HTML/CSS/JavaScript
